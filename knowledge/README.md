# 知识库目录

> **用途**: 沉淀项目开发中遇到的难题、解决方案、最佳实践和架构模式，供日后快速参考。

---

## 📋 使用规范

### **何时参考知识库？**

**优先查阅**的场景：

- ✅ 遇到类似的技术难题
- ✅ 需要实现类似的功能模式
- ✅ 性能优化需求
- ✅ 排查疑难Bug

**建议浏览**的场景：

- 🟡 新成员熟悉项目
- 🟡 技术方案设计前

### **何时新增知识文档？**

**必须沉淀**的场景：

- ✅ 解决了耗时超过2小时的难题
- ✅ 发现了框架/库的非常规用法
- ✅ 总结了可复用的架构模式
- ✅ 性能优化效果显著（提升>30%）

**可选沉淀**的场景：

- 🟡 常见问题的标准解法
- 🟡 第三方库的最佳实践

### **知识文档命名规范**

```
格式: 描述性名称.md (使用中划线分隔)

示例:
- troubleshooting/websocket-reconnect-failure-solution.md
- patterns/dynamic-table-column-configuration.md
- performance/large-dataset-rendering-optimization.md
```

### **知识文档模板**

````markdown
# [主题] 标题

## 📝 问题描述

简述遇到的问题或需要实现的功能。

## 🔍 根本原因 / 核心需求

深入分析问题根源或需求本质。

## 💡 解决方案

### 方案概述

一句话总结解决思路。

### 技术实现

```code
// 关键代码示例
```
````

### 关键要点

- 要点1: 为什么这样做
- 要点2: 注意事项

## ⚠️ 注意事项

- 适用场景
- 不适用场景
- 潜在风险

## 📚 相关资源

- 官方文档链接
- 相关知识库文档
- 原始方案文档（如有）

## 📅 文档元信息

- 创建日期: YYYY-MM-DD
- 最后更新: YYYY-MM-DD
- 关联项目版本: vX.X.X

